# 명세서: `tests/unit/test_expand_vars.py`

## 역할
- `common.expand_vars.expand_vars` 함수가 문자열/딕셔너리/리스트/복합 구조/기타 타입에 대해 기대한 대로 환경 변수 치환을 수행하는지 검증한다.

## 테스트 범위 (요약)
- 문자열: 단일/복수/반복 변수 치환, 정의되지 않은 변수 유지, 빈 문자열 처리.
- 딕셔너리/리스트/중첩 구조: 트리 전체에 대한 재귀적 치환과 로깅 경로(`LOG_PATH`) 확장을 검증한다.
- 기타 타입: `int`, `float`, `bool`, `None` 등이 그대로 반환되는지 확인한다.
- 불변성: 입력 딕셔너리가 변경되지 않고 새 딕셔너리가 반환되는지 검증한다.

## 핵심 규칙
- 각 테스트 전후로 관련 환경 변수를 설정/정리하여 테스트 간 독립성을 유지한다.
- 복합 구조 테스트에는 GS/ISMS-P 마커를 사용해 시험 가능성과 보안 요구사항(환경 기반 설정 구조)을 증적한다.
